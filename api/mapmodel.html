Map Object Model

This is the object model for a Map object created with dusk.buildMap. Please note that this page does not include functions, but rather only properties.

    Dusk Map

  • map.props : Properties Table

      • map.layer

    • Contains map layers from Tiled.
    • map.layer[index] : Layer object built in Tiled. Index 1 is in the front of the map.
    • map.layer[name] : Layer object built in Tiled. name corresponds to the name of a layer in Tiled.

      map.data

    • Contains map data values.
    • map.data.numLayers : Number of layers loaded in the map.
    • map.data.numTiledLayers : Number of layers built in Tiled. This may not be the same as .numLayers, on account of Dusk's !inactive! property.
    • map.data.tilesetCount : Number of tilesets in the map.
    • map.data.mapWidth : Width of map in tiles.
    • map.data.mapHeight : Height of map in tiles.
    • map.data.rawTileWidth : Map tile width before scaling, equal to what was set in Tiled.
    • map.data.rawTileHeight : Map tile height before scaling, equal to what was set in Tiled.
    • map.data.tileWidth : Map tile width after Dusk's automatic scaling is applied.
    • map.data.tileHeight : Map tile height after Dusk's automatic scaling is applied.
    • map.data.width : Width of map in pixels. This may not be the same as Corona's normal map.width, on account of the effect of tile culling.
    • map.data.height : Height of map in pixels. This may not be the same as Corona's normal map.height, on account of the effect of tile culling.
