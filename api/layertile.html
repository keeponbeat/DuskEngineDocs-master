layer.tile()

Description:

  • Returns the tile at an XY position in tiles.

Syntax:

  • layer.tile(x, y)

Arguments:


  • x (XY Pair-X, required) : The X-coordinate (in tiles) of the tile desired.
  • y (XY Pair-Y, required) : The Y-coordinate (in tiles) of the tile desired.

Returns:


  • tile (Tile) : The tile at the XY position. If no tile exists, this will be nil.

Example:


local tile = map.layer[1].tile(10, 10)
