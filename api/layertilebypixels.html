layer.tileByPixels()

Description:

  • Returns the tile at an XY position in pixels. Pixels are given in local coordinates for the layer.

Syntax:

  • layer.tileByPixels(x, y)

Arguments:


  • x (XY Pair-X, required) : The X-coordinate (in pixels) of the tile desired.
  • y (XY Pair-Y, required) : The Y-coordinate (in pixels) of the tile desired.

Returns:


  • tile (Tile) : The tile at the XY position. If no tile exists, this will be nil.

Example:


local tile = map.layer[1].tileByPixels(512, 384)
