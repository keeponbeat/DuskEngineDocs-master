Properties Table

A Properties Table is a data member that holds properties from the Tiled properties dialog. It does not hold every property entered; only those prefixed with props:. This is done so that "special" values such as alpha, x and y, width, et al may be added to the object without Corona interpreting them as display object parameters. Values added to this table are retrieved from the object's props table.

If an object is listed as having a Properties Table, this means it has this data member and accepts values prefixed with props:.

Example:

-- Created a map with a layer having "props:alpha" as 500
print(map.layer[1].props.alpha) --> 500
