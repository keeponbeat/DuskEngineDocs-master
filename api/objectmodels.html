Object Object Models

These are the object models for the objects within an Object Layer. Objects may be accessed with indices, names, or iterators. The word "Object" is not meant in a generic sense and is used (when unmodified) to refer to objects created by Tiled within an Object Layer.

    Object

  • object.props : Properties Table
  • object._objType : String equal to the type of object it is.
  • object._name : Corresponds to the value set in Tiled's "Name" field within the object's properties dialog. This value can be used to access the object within the object's parent layer; each object is stored in the parent layer's object table as layer[object._name].
  • object._type : Corresponds to the value set in Tiled's "Type" field within the object's properties dialog.

      • Ellipse

    • An ellipse object is created with the ellipse drawing command within Tiled. They represent any oval shape.
    • An ellipse object's _objType is ellipse.

      Polygons & Polylines

    • Polygon and polyline objects are created with the polygon and polyline drawing commands within Tiled. They represent vertices connected with lines. The only difference betwixt the two is that a polygon object has one additional line reaching from the last vertex to the beginning vertex.
    • A polygon object's _objType is polygon, and a polyline's _objType is polyline.
    • object.points : The vertices of the polygon or polyline. These are given in the form points[i] = {x = x, y = y} where i is the vertex index, x is the X-coordinate of the vertex, and y is the Y-coordinate of the vertex. The origin for these vertices is the polygon or polyline's first vertex.

      Image Objects

    • Image objects are created with the image stamp tool within Tiled. They are created as single tiles, yet they are more customizable than a Tile Layer's tiles. An important thing to note, however, is that image objects are not culled.
    • An image object's _objType is image.

      Rectangle Objects

    • Rectangle objects are created with the rectangle drawing tool within Tiled. They represent a four-sided polygon with four right angles and two pairs of parallel sides.
    • A rectangle object's _objType is, by default, dependent on its dimensions. To disable this behaviour, set the objTypeRectPointSquare preference to false (see this guide for more information).
    • A rectangle whose dimensions are (w, h) = (0, 0) has by default an _objType of point, a rectangle with dimensions w = h has by default an _objType of square, and everything else has an _objType of rectangle. If the objTypeRectPointSquare setting is disabled, all rectangle objects have an _objType of rectangle.
