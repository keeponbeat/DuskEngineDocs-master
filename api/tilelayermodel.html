Tile Layer Object Model

This is the object model for a Tile Layer object created from within dusk.buildMap and existing in a Dusk Map. Please note that this page does not include functions, but rather only properties.

    Dusk Tile Layer

  • layer.props : Properties Table
  • layer._type : Equal to the string "tile". This property is given to all layers in a map; by having one layer, you can know what kind it is.
  • layer._name : The name specified in Tiled, found in the layer listing menu.
  • layer.xParallax : Parallax ratio of the layer in the X-direction.
  • layer.yParallax : Parallax ratio of the layer in the Y-direction.
  • layer.cameraTrackingEnabled : Whether this layer moves with the camera's tracking. This property does not have any effect if set after the map is created; if you do not want the layer to move with the camera, set this property from within Tiled.
  • layer.tileCullingEnabled : Whether this layer's tiles are culled. This property does not have any effect if set after the map is created; if you do not want the layer's tiles to be culled, set this property from within Tiled.
