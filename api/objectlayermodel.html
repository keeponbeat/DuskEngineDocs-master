Object Layer Object Model

This is the object model for an Object Layer object created from within dusk.buildMap and existing in a Dusk Map. Please note that this page does not include functions, but rather only properties.

    Object Layer

  • layer.props : Properties Table

      • layer.object

    • Contains objects created within the layer.
    • layer.object[index] : Object built in Tiled.
    • layer.object[name] : Object built in Tiled. name corresponds to the name of an object in Tiled (set within the "Name" field of an object's properties).
