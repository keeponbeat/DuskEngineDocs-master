Object Iterators

Dusk provides a number of interfaces for accessing objects within an object layer. All iterators return the current iterated object for each iteration.

Format

for object in map.layer[k].<iterator>() do
	-- Iterate
end

Iterators

      layer.nameIs(n)

    • Iterates through all objects whose name is [n]. The name corresponds to what was entered into the Tiled "name" field for the object.

      layer.nameMatches(n)

    • Iterates through all objects whose name matches [n], using Lua patterns. The name corresponds to what was entered into the Tiled "name" field for the object.

      layer.typeIs(n)

    • Iterates through all objects whose type is [n]. The type corresponds to what was entered into the Tiled "type" field for the object.

      layer.typeMatches(n)

    • Iterates through all objects whose type matches [n], using Lua patterns. The type corresponds to what was entered into the Tiled "type" field for the object.

      layer.objTypeIs(n)

    • Iterates through all objects whose object type is [n]. The object type of an object corresponds to the way the object was created in Tiled. See this guide (the _objType property) for more information.

      layer.objects()

    • Iterates through all objects in the layer.
