map.updateView()

Description:

  • Updates the map's viewport. If the camera system is enabled, each layer built with the camera enabled will be updated with the camera. If the culling system is enabled, each tile layer built with culling enabled will be updated with culling. If both are enabled, both will be updated.

Syntax:

  • map.updateView()

Arguments:


  • (none)

Returns:


  • (none)

Example:


map.updateView()
