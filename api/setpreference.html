dusk.setPreference()

Description:

  • Sets the value of a library preference. See the preference guide for information on preference values.

Syntax:

  • dusk.setPreference(preferenceName, value)

Arguments:


  • preferenceName (string, required) : The name of the preference to set. Names are case-sensitive.
  • value (preference type, required) : The new value of the preference. This must be the type of the preference; Boolean preferences may not be set to numbers, for example.

Returns:


  • (none)

Example:


dusk.setPreference("enableCamera", true)
