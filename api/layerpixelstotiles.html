layer.pixelsToTiles()

Description:

  • Converts pixel coordinates to tile coordinates. The coordinates you provide are used as local coordinates for the layer.

Syntax:

  • layer.tilesToPixels(x, y)

Arguments:


  • x (XY Pair-X, required) : The X-coordinate (in pixels) to be converted.
  • y (XY Pair-Y, required) : The Y-coordinate (in pixels) to be converted.

Returns:


  • x (number) : The converted X-coordinate.
  • y (number) : The converted Y-coordinate.

Example:


local tileX, tileY = map.layer[1].pixelsToTiles(512, 384)
