dusk.buildMap()

Description:

  • Generates a map object according to a data file exported from Tiled.

Syntax:

  • dusk.buildMap(filename, [base])

Arguments:


  • filename (string, required) : The path to the map file, including extension. Directories are separated with forward slashes (/). Dusk supports either JSON or Lua encoding for map files, both available from Tiled's export menu.
  • base (Corona directory constant, optional) : The base directory of the map file. This must be a Corona directory constant, to wit, system.DocumentsDirectory, system.CachesDirectory, system.TemporaryDirectory, or system.ResourceDirectory. Defaults to system.ResourceDirectory. Does not apply to Lua maps.

Returns:


  • map (Map) : The resulting map.

Example:


local map = dusk.buildMap("directory/mapname.lua")
